Página Web con Bootstrap 5

Diseño moderno, responsivo y personalizado con CSS3

Perfil Académico

Somos estudiantes del área de Tecnologías de la Información, interesados en el desarrollo web y en la aplicación de herramientas modernas como Bootstrap y CSS3, con el objetivo de crear interfaces funcionales, accesibles y visualmente atractivas.

Uso de Bootstrap 5

En esta sección se evidencia el uso del framework Bootstrap 5 para estructurar el contenido mediante su sistema de grid y componentes predefinidos, permitiendo un diseño ordenado, consistente y adaptable a distintos dispositivos.

Diseño Responsivo

El sitio web ha sido desarrollado bajo principios de diseño responsivo, asegurando su correcta visualización en computadoras, tablets y teléfonos móviles, gracias al uso de clases responsivas y media queries personalizadas.

Este contenido se muestra y se oculta usando el componente collapse de Bootstrap 5, demostrando interactividad y animaciones suaves.
